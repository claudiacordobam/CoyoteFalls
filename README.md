# 🎯 CoyoteFalls

![License](https://img.shields.io/badge/license-GPLv3-blue)  
![Processing](https://img.shields.io/badge/Processing-4.2-orange)

Simulate **Wile E. Coyote** falling in **classic cartoon style** vs **realistic physics**.  
Compare fun cartoon physics with real-world gravity! 🌟

---

## 🔍 About

This project visualizes the difference between cartoon and real-world physics:  

- **🖼 Cartoon-style:** Horizontal run, then sudden free-fall – just like the classic cartoons!  
- **📐 Realistic:** Parabolic trajectory under gravity – what would really happen.  

Perfect for learning physics in a fun and interactive way! 🎢

---

## ✨ Features

- Two simulation modes: `Cartoon` & `Realistic`  
- Visual representation of trajectories  
- Easy to run in Processing  

---

## 🚀 How to Run

1. Clone or download the repository:

```bash
git clone https://github.com/claudiacordobam/CoyoteFalls.git
```
2. Open the file in Processing
3. Press Run to start the simulation.

---

##🎬 Examples

**Cartoon Mode** 🖼️
The Coyote runs horizontally and then falls abruptly.

**Realistic Mode** 📐
The Coyote falls along a parabolic trajectory under gravity.
